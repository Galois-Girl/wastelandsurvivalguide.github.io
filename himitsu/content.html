Wasteland Survival Guide

Content

First, create a "Content" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
The Living Desert Adds several scripted events & hundreds of NPCs who will patrol roads, travel between towns, occupy locations, explore the desert, and react dynamically to your choices throughout the game
  • Main File - The Living Desert
  • AWOLP/Living Desert Compatibility Patch - Underpass Patch - All Mods
The Someguy Series LE This master file allows you to play mods in the Someguy series, which are being remastered. There are currently two episodes available
  • Main File - Someguy Series 2 dot 0
  • For each episode, install the Main File and Optional File - TTW Patch:
    • New Vegas Bounties I LE
    • Bad Mothafucka LE
Uncut Wasteland & Extra Collection Restores a huge amount of scenery, NPCs, creatures and other details which were patched out New Vegas post-release. These versions are patched for TTW and FPGE by Kazopert
  • Main File - Uncut Wasteland
  • Main File - Uncut Extra Collection
  • Patch - FPGE - Uncut Wasteland & Extra Collection Patch
Vikki and Vance Improvements Improve some aspects of the Vikki & Vance casino so it actually looks like a casino while staying true to the vanilla aesthetic Main File - Vikki and Vance Improvements
