Wasteland Survival Guide

Content

First, create a "Content" separator in MO2.

As you install the mods from this section, organize them underneath this separator.
A World of (Less) Pain
A Lore Friendly AWOP Revision 		An edit of the original A World of Pain that removes all unvoiced NPCs, quests, un-lore friendly interiors, and "M" weapons, ending up with about 60 rebalanced locations Main File - AWOP Revised
AWOP4FO3
TTW A World of Pain for Fallout 3 		Adds many intense and unique locations to the Capital Wasteland, new items, 4D storage, two companions and a couple of quests.
  • Main File - AWOP4FO3
  • Optional File - JIP CCC Avatars
  • Optional File - TTW AWOP4FO3 - No Starter Terminal
    Wasteland Starting Gear already provides starting loadouts and the 4D storage is overpowered; that is why the terminal is suggested to be disabled
  • Patch - AWOP FO3 - TTW Patch
Casino Crowds Increased NPC population in all 3 Casinos: 80+ across Gomorrah, 55+ across The Tops, 15+ in The Ultra-Luxe
  • Main File - Casino Crowds
  • Patch - Casino Crowds - TTW Patch

    • Main File - Mojave Raiders

    • TTW - Mojave Raiders Patch
    Mojave Squads Adds a "Mercenary" system to FNV allowing you to hire soldiers from 20 factions with over 150 choices to choose from. Includes an Optional Addon for TTW
    • Main File - Mojave Squads
    • Main File - Mojave Squads Config File

    Rebuild the Capital
    • Expand the Brotherhood of Steel and their role in the wasteland following the events of Broken Steel
    • Under the player's guidance, the Brotherhood can establish relations with settlements and establish military outposts
    • The mod also features new armors, weapons, quests, perks, NPCs, and a recruitment system
    • Main File - Rebuild the Capital
      • In the FOMOD check No Pony Express Boxes, then click
    • Update File - Rebuild the Capital Hotfix
    • Optional File - PAVE Patch
    • Patch - TTW - ROOG - Rebuild the Capital Patch
    Simple Open Strip Modern recreation of The Strip Open with mesh/bugfixes & occlusion planes for stability and performance
    • Main File - Simple Open Strip
    • Optional File - Simple Open Strip - Vanilla LOD
      • If you use texture packs, you will need to avoid this file and generate your own LOD
    Sweet Home Explore an old rivalry between a notorious terrorist of the Wastes and an old ghoul that seeks vengeance. Make a fateful decision which could shape the Wastes forever. What will you decide? Who will live? Who will you believe? Main File - Sweet Home
    The Living Desert Adds several scripted events & hundreds of NPCs who will patrol roads, travel between towns, occupy locations, explore the desert, and react dynamically to your choices throughout the game
    • Main File - The Living Desert
    • AWOLP/Living Desert Compatibility Patch - Underpass Patch - All Mods
    The Someguy Series LE This master file allows you to play mods in the Someguy series, which are being remastered. There are currently two episodes available
    • Main File - Someguy Series 2 dot 0
    • For each episode, install the Main File and Optional File - TTW Patch:
      • New Vegas Bounties I LE
      • Bad Mothafucka LE
    Transporters TTW Take on the Wastelands with a new alternative form of Fast-Travel, with over 37+ travel destinations!

    • Patch - SD Transporters - AWOP FO3 Patch
