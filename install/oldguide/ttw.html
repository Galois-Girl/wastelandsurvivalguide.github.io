Installing Tale of Two Wastelands

In this step, we will install and activate Tale of Two Wastelands through Mod Organizer 2.

Downgrading Fallout 3
This step is only necessary if you have Fallout 3 installed from Steam.
  1. Make sure both Mod Organizer 2 and Fallout 3 are closed
  2. Download the Fallout 3 Downgrader file
  3. Extract the contents of the Fallout3Downgrader archive to the Root of Fallout 3
  4. Run Fallout3Downgrader.exe

Installing TTW
If you run into any errors when installing, check the FAQ to see if it has the solution.
  1. Make sure both Mod Organizer 2 and the game are closed
  2. From the TTW Download page, download the 3.2 Installer, 3.2.1 Incremental Update, and 3.2.2 Incremental Update files
  3. Extract the contents of the 3.2. Installer archive to a new folder called TTW Installer in a safe location outside of any default Windows folders
If you do know what is meant by "default Windows folders", read the Installation Location section from the Initial Setup page.
  1. Extract the contents of both the 3.2.1 Incremental Update and 3.2.2 Incremental Update archives to the same folder
  2. Once everything has been extracted, navigate to Mod Organizer 2\mods\ and create a folder there named Tale of Two Wastelands
If you are unsure as to where/what your MO2 Mods folder is, open up MO2, select the button at the top, and select Open Mods folder
  1. Back in the TTW Installer folder, right-click the TTW Install.exe and select Run As Administrator
  2. From the pop-up, select TTW_v3.2_Full then Install
  3. In the installer, verify that the file paths for both your Fallout 3 and Fallout New Vegas installations are correct
  4. Set the Tale of Two Wastelands install path to the Mod Organizer 2\mods\Tale of Two Wastelands you created in step 5
  5. Select Install
The install will take a very long time (can be anywhere from 2-12 hours depending on your CPU, it took 3 hours on my i5 7500). It is recommended to leave your computer be and have any CPU-intensive applications closed during this time as to not overheat your CPU and cause the TTW installation to fail.
  1. Once the installer has finished, exit it and immediately re-run it as an administrator
  2. This time, select TTW_v3.2_to_v3.2.1_Update then Install
  3. In the installer, verify that the file paths for your Fallout New Vegas and Tale of Two Wastelands 3.2 installations are correct
  4. Set the install path for Tale of Two Wastelands 3.2.1 to the same folder that you installed Tale of Two Wastelands 3.2 to (Mod Organizer 2\mods\Tale of Two Wastelands)
  5. Select Install
  6. Once the installer has finished (should only take a few seconds), exit it and immediately re-run it as an administrator
  7. This time, select TTW_v3.2.1_to_v3.2.2_Update then Install
  8. In the installer, verify that the file paths for your Fallout New Vegas and Tale of Two Wastelands 3.2.1 installations are correct
  9. Set the install path for Tale of Two Wastelands 3.2.2 to the same folder that you installed Tale of Two Wastelands 3.2.1 to (Mod Organizer 2\mods\Tale of Two Wastelands)
  10. Select Install
  11. Once the installer has finished (should only take a few seconds), exit it

Activating TTW in Mod Organizer 2
  1. Launch Mod Organizer 2
  2. Make sure the The Best of Times profile is active
  3. If you installed TTW correctly with the correct filepaths, you should see the Tale of Two Wastelands mod in the left pane of MO2
    • If you do not see anything, you did not install TTW to the correct folder. Find out where the files were installed to and move them to Mod Organizer 2\mods\Tale of Two Wastelands
  4. Click the checkbox next to it in the left pane, and the right pane should fill with plugins
You can now safely uninstall Fallout 3.
