Gameplay

Creating a Separator in MO2
  1. Right-click anywhere in the left pane of MO2 and select All Mods -> Create Separator
  2. Name the separator Gameplay

This page contains mods that greatly overhaul the game by making it much more challenging (and more rewarding in turn) and adding new features. These mods are not necessarily recommended for a first playthrough of the game, and are 100% optional. It is especially important to read the descriptions of these mods (you should still read the description of every mod), so you know what has changed and how to use the new features.

JIP LN Overhauls INI

Installation instructions:

  • Download and install normally through MO2

- Allows NPCs to use weapon modifications and enables Fallout 3-style repair in which weapons can only be repaired up to a condition % equal to your repair skill

lStewieAl's Tweaks Overhauls INI

Installation instructions:

  • Download and install normally through MO2
IMPORTANT: It is HIGHLY RECOMMENDED to read through the two INI files from this. They contain relatively major gameplay changes and you will be very confused if you don't. It's only a few lines and takes less than a minute to read.

- Enables extra gameplay-oriented settings in Stewie's Tweaks without completely overwriting the original INI

Delay DLC TTW

Installation instructions:

  • Main File - Delay DLC 1.0 - TTW 3.2 1.0

- stops the annoying messages that display when a new DLC is loaded, and requires that you be within a certain distance of the broadcast location before you can pick up the radio signal.

Once you do have the signal, you have to listen to the entire thing in order to get the quest and map marker for the location. Do not report this as a bug!

sawyerbatty

Installation instructions:

  1. Follow the Google Drive link and download the mod using the download button pictured below
  2. Install the mod normally through MO2

- A variation on JSawyer, the gameplay re-balance mod made by the lead director of the game made, edited for TTW. The original JSawyer changelog is here

Fallout 4 Survival Mode-Style Damage

Installation instructions:

  • Main File - Fallout 4 Survival Damage 2.0

- Changes damage multipliers to make combat more difficult and adjusts the weak spots of some enemies to make every encounter feel unique

With the previous two mods installed, it is highly recommended to play on High difficulty or lower.

Misc. Gamesetting Tweaks

Installation instructions:

  • Download the Main File - Misc. Gamesetting Tweaks Presets 5.3 and install normally through MO2
  • Select any presets you would like to enable. If you change your mind, you can reinstall the mod and revise your selections
It is up to you which (if any) tweaks to enable. "Bullet Impact Increased LOD", "Faster Terminal Speed", "No Auto-Aim", "Perk Every Level" & "Weapon Follows Crosshair in First Person" are some suggestions. Some of these are already enabled by the Tweaks INIs we've installed; this will be mentioned in the installer.

- A collection of INI mods to configure Game Settings to your liking. Powered by lStewieAl's Tweaks.

Fallout 3 Economy

Installation instructions:

  • Main File - Fallout 3 Economy 2.0.0

- Overhauls item values to be similar to Fallout 3's much tougher economy

Follower Tweaks

Installation instructions:

  • Main File - Follower Tweaks 1.0

- Reworks Boone, Ed-E, and Rexes' companion perks to be less intrusive/annoying

Just Assorted Mods

Installation instructions:

  • Main File - Just Assorted Mods 4.3c

- Highly configurable set of new modern features such as a quick loot menu, dynamic crosshair, sprinting, and more

Just Assorted Mods Custom INI

Installation instructions:

  • Download and install normally through MO2

- Enables/disables/tweaks certain settings to be as balanced and true to vanilla as possible. Can be further tweaked to your liking in the Mod Configuration Menu
